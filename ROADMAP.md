# Roadmap — Islands-Enet

Plan de desarrollo de 5 meses (~22 semanas). Cada mes tiene un foco principal, pero el contenido de categorías anteriores sigue sumándose en paralelo a menor ritmo.

## Mes 1 — Cimientos técnicos
**Foco: Entendimiento de Java y coordinacion mod-datapack**
- [x] Setup del entorno (Fabric 1.16.5, Gradle, VS Code)
- [x] Repositorio en GitHub configurado (README, LICENSE, .gitignore)
- [ ] Registro de bloques (patrón base reutilizable)
- [ ] Registro de entidades (patrón base reutilizable)
- [ ] Pipeline datapack ↔ mod funcionando
- [ ] Sistema de admin — panel básico (`tellraw` + tags)
- [ ] Primer bloque de prueba end-to-end (ej: Mena de Pirita)

## Mes 2 — Contenido núcleo
**Foco: Bloques + Entidades + Items fundacionales**
- [ ] Bloques base por bioma/isla (minerales, terreno especial)
- [ ] Entidades base (mobs comunes de cada tipo de isla)
- [ ] Items fundacionales (herramientas/materiales que se usan en el resto del proyecto)
- [ ] Sistema de admin — scoreboards en vivo (jugadores online, tick del día)

## Mes 3 — Estructuras y worldgen
**Foco: Estructuras**
- [ ] Estructuras especiales por tipo de isla
- [ ] Worldgen (generación de islas, biomas)
- [ ] Loot tables de estructuras
- [ ] Sistema de admin — botón "Invocar estructura" funcional

## Mes 4 — Jefes y misiones
**Foco: Jefes + Misiones**
- [ ] Jefes (mecánicas de combate, fases)
- [ ] Misión principal — diseño de progresión
- [ ] Misiones secundarias
- [ ] Misiones diarias
- [ ] Sistema de admin — botón "Probar mobs" funcional

## Mes 5 — Pulido y release
**Foco: Ajustes de dificultad y pruebas finales**
- [ ] Balance general (números, dificultad)
- [ ] Bugs y testing
- [ ] Textos, sonidos, traducciones
- [ ] Sistema de admin — panel completo y pulido
- [ ] Release candidate
